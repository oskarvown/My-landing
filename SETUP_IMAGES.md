# Инструкция по добавлению изображений

## Необходимые файлы

Для корректной работы секции "Обо мне" необходимо добавить следующие файлы:

### 1. Фотография профиля

**Путь:** `assets/images/askar-photo.jpg`

**Требования:**
- Формат: JPG, PNG или WebP
- Рекомендуемый размер: 400x400 пикселей (квадратное изображение)
- Качество: высокое, четкое фото лица

**Как добавить:**
1. Создайте папку `assets` в корне проекта
2. Создайте папку `images` внутри `assets`
3. Сохраните вашу фотографию как `askar-photo.jpg` в `assets/images/`

### 2. Резюме (опционально)

**Путь:** `assets/askar-resume.pdf`

**Требования:**
- Формат: PDF
- Размер: до 5 МБ

**Как добавить:**
1. Сохраните PDF-файл резюме как `askar-resume.pdf` в папку `assets/`

## Структура директорий

После добавления файлов структура проекта должна выглядеть так:

```
d:\Сайт Портфолио Аскар\
├── assets/
│   ├── images/
│   │   └── askar-photo.jpg
│   └── askar-resume.pdf
├── src/
├── index.html
└── ...
```

## Временное решение

Если у вас пока нет готовой фотографии, можно использовать placeholder:
- Раскомментируйте строку с unsplash в `About.jsx`
- Или используйте любое другое изображение

---

**Примечание:** Фотография будет отображаться с градиентной рамкой (фиолетовый → розовый → оранжевый) в круглой форме размером 192x192 пикселя.
